Hey folks it's my first web page. Here you can paly Snake game for now. This snake game is only for single player and can be played by the left, right, up, down arrow keys. The snake has to eat the food present on the board. By eating the food the snake increases it's length and the user can know about the score on top left corner of the board. The snake can die if it crash into itself and it's Game Over. There is no further lives. So be careful while playing. In future I will advance the snake game and also can add other intresting games.
